📄 Project Report: AI-Based Autonomous Navigation System
👤 Student Name: Vaidehi Dhiraj Pawar
🛠️ Project Overview:
I have built a 2D simulation that mimics the decision-making of an autonomous vehicle. It uses the A (A-Star) Algorithm* to find the shortest path between a start point and an end point while avoiding obstacles.

⚙️ System Architecture:
Perception: Identifies obstacles and goal coordinates.
Path Planning: Uses Heuristic Search to calculate the most efficient route.
Execution: The agent moves along the calculated path in real-time.
🚀 Future Scope:
Integrating real-time camera input using OpenCV.
Implementing dynamic moving obstacles.
Upgrading to a 3D simulation using CARLA or ROS.
